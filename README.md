# Client-Side Transaction Analysis App

Vue 3 + TypeScript + Vite app for importing a standardized `.xlsx` transaction file, validating/normalizing rows, and analyzing data fully in-browser.

## Highlights

- Pure client-side processing with SheetJS
- Header auto-detection by required schema
- Validation + row-level error collection
- Sankey dashboard via Apache ECharts
- Global filters and KPI cards
- Virtualized editable data grid
- localStorage persistence + JSON export

## Run

```bash
npm install
npm run dev
```
